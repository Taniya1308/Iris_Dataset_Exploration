# Iris Dataset Exploration

This project explores the Iris dataset using Python.

## Tools Used

- NumPy
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed

- Data exploration
- Statistical analysis
- Pair plots
- Feature relationships

## Dataset

Iris dataset containing measurements of iris flowers:

- Sepal length
- Sepal width
- Petal length
- Petal width
- Species
